Software

Open-source tools

OptiQ software focuses on robust workflows for noisy, hybrid quantum–classical computation: benchmarking quantum ML models beyond accuracy, and variational quantum algorithms for quantum chemistry.

SA-OO-VQE (Qiskit)

A state-averaged, orbital-optimized VQE workflow designed to treat ground and excited electronic states on a comparable footing. The package is implemented in Python on top of Qiskit, with documentation and distribution via PyPI.

  • Focus: VQAs for quantum chemistry, state-averaging, orbital optimization
  • Distribution: pip installable package (saoovqe)

Repository (GitLab)  •  PyPI  •  Documentation  •  Paper (arXiv:2401.11884)  •  PDF

QMetric

A benchmarking toolkit for quantum neural networks and hybrid quantum–classical models. QMetric provides interpretable diagnostics that complement raw accuracy by exposing issues in circuit design, feature encoding, and training dynamics.

  • Focus: QML evaluation, expressibility, training stability
  • Use cases: comparing circuits/encodings, diagnosing training failures

Paper (arXiv:2506.23765)  •  Repository (GitLab)
